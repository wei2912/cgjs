# Operators
## Assignment Operator ```js var a = 0; a = a + 10; var firstName, lastName; // assign to several variables firstName = lastName = ""; ```
## Operator Precedence ```js 3 + 30 / 2 - 3 * 3; // 3 + 15 - 9 = 9 (3 + 30) / (2 - 3) * 3; // 33 / (-1) * 3 = -99 // TL;DR: It’s math. ```
## Assignment with Operation ```js var number = 50; number += 3; // number = number + 3; number -= 10; // number = number - 10; number *= 2; // number = number * 2; number /= 3; // number = number / 3; number %= 5; // number = number % 5; ```
## Equality/Inequality Operators ```js 3 == 3; // true 5 == 3; // false 'a' == 'a'; // true 'ab' == 'a'; // false 3 != 3; // false 5 != 3; // true 'a' != 'a'; // false 'ab' != 'a'; // true ```
## Logical Operator — AND ```js expr1 && expr2; var value = 10; (value > 10) && (value <= 100); // (value > 10) AND (value ≤ 100) // false ```
## Logical Operator — OR ```js expr1 || expr2; var value = 10; (value > 10) || (value <= 100); // (value > 10) OR (value ≤ 100) // true ```
## Logical Operator — NOT ```js !expr; var number = 10; number < 10; // false !(number < 10); // true ```
