# Branches
## `if` ```js if (condition) { "statements to be processed if condition is truthy"; } ```
## `if` ```js var number = 5; if (number < 10) { alert('number is less than 10'); } ```
## `if…else` ```js if (condition) { "statements to be processed if condition is truthy"; } else { "statements to be processed if condition is falsey"; } ```
## `if…else` ```js var number = 5; if (number < 10) { alert('number is less than 10'); } else { alert('number is 10 or greater'); } ```
## `if…else` ```js var number = 15; if (number < 10) { alert('number is less than 10'); } else if (number < 20) { alert('number is 10 or greater but lesser than 20'); } else { alert('number is 20 or greater'); } ```
## Ternary operator (the only one) ```js condition ? "value if truthy" : "value if falsey"; var number = 10; alert( (number > 10) ? "number over 10" : "number not over 10"; ); ```
## `switch` ```js switch (expression) { case firstLabel: // statements break; case secondLabel: case thirdLabel: // statements break; default: // statements break; // optional } ```
## `switch` ```js var abbreviation, name; switch (abbreviation) { case 'wei2912': name = 'Ng Wei En'; break; case 'demoneaux': case 'd10': name = 'Benjamin Tan'; break; default: name = 'Unknown'; break; } ```
# Loops
## `while` ```js while (condition) { statements; } ``` 1. If `condition` is true, proceed to step 2. Otherwise, end the loop. 2. Execute `statements`. 3. Return to step 1 again. Every run of this process is called an iteration.
## `while` ```js // declare variables first var i = 1, noOfTimes = 10; while (i <= noOfTimes) { alert("Alert no. " + i); ++i; } ```
## `do…while` ```js do { statements; } while (condition); ``` 1. Execute `statements`. 2. If `condition` is true, return to step 1. Otherwise, end the loop.
## `do…while` ```js // declare variables first var i = 1, noOfTimes = 10; do { alert("Alert no. " + i); ++i; } while (i <= noOfTimes); ```
## `for` ```js for (init; condition; update) { statements; } ``` 1. Execute `init`. 2. If condition is truthy, proceed to step 3. Otherwise, end the loop. 3. Execute `statements`. 4. Execute `update`. 5. Return to step 2.
## `for` ```js for ( var i = 1, noOfTimes = 10; // init i <= noOfTimes; // condition ++i // update ) { alert('Alert no. ' + i); } ```
## `continue` ```js for (init; condition; update) { continue; } while (condition) { continue; } do { continue; } while (condition); ``` `continue` skips the current execution of `statements` and goes to the next iteration.
## `continue` ```js // Alert from 0 up to 100 … for (var i = 0; i < 100; ++i) { // … but don’t alert multiples of 13 if (i % 13 == 0) { continue; // … skips to ++i, then i < 100, … } else { alert(i); } } ```
## `break` ```js for (init; condition; update) { break; } while (condition) { break; } do { break; } while (condition); ```
## `break` ```js // Alert from 0 up to 100 … for (var i = 0; i < 100; ++i) { alert(i); // … but end at the first multiple // of 13 encountered if (i % 13 == 0) { break; } } ```
## Labeled loops ```js outer: while (condition) { for (init; condition; update) { if (/* … */) { break outer; // exit both inner and outer loops } else { continue outer; // skip to next iteration of outer loop } } } ```
# Objects
## Objects * Storage containers * Associate values with keys * Can contain anything
## Creating Objects ```js var object = { 'x': 'Hello', 'y': 'Hello2' }; ```
## Modifying Objects ```js var object = { 'x': 'Hello' }; object.x = 'Hello World'; object.y = 100; delete object.y; object.y; // undefined object.nonExistentKey; // undefined ```
## Comparing Objects ```js var object = {}; object == {}; // false object == object; // true ```
# Functions
## Functions * Objects * Encapsulate pieces of code * Make it easier to program * 1 per action
## Creating Functions ```js // function declaration function sayHi(name) { alert('Hi ' + name + '!'); } sayHi('Benjamin'); // note: function declarations **cannot** be // used reliably in branches/loops ```
## Creating Functions ```js // function expression var sayHi = function(name) { alert('Hi ' + name + '!'); }; sayHi('Jared'); // note: function expressions **can** be // used reliably in branches/loops ```
## Function Scope ```js var name = 'name'; function test() { var name = 'Benjamin'; console.log(name); } test(); // logs 'Benjamin' console.log(name); // 'name' ```
## Returning Values ```js function noop() {} noop(); // undefined function sayHi(name) { return 'Hi ' + name + '!'; } sayHi('Wei En;); // 'Hi Wei En!' ```
## Arguments * Primitives are passed into functions as values * Objects are passed into functions by reference
## Arguments ```js var number = 10; var object = {}; function modifyArguments(num, obj) { num = 1000; obj.x = 'x'; } modifyArguments(number, object); console.log(number); // 10 console.log(object.x); // 'x' ```
