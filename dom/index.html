# Introduction to DOM
## What is the DOM? Document Object Model * Representation of the HTML tree in JavaScript
## Creating elements ```js var divElem = document.createElement('div'); var scriptElem = document.createElement; ```
## Modifying attributes ```js divElem.id = 'shiny-div'; // sets the "class" attribute divElem.className = 'special shiny'; scriptElem.src = '/path/to/file.js'; // or divElem.setAttribute('id', 'shiny-div'); divElem.setAttribute('class', 'special shiny'); scriptElem.setAttribute('src', '/path/to/file.js'); ```
## `classList` ```js divElem.classList.add('class'); divElem.classList.remove('useless-class'); divElem.classList.toggle('visible'); ```
## `innerHTML` ```js divElem.innerHTML = 'This is a paragraph of text containing ' + ' a useless link.'; ```
## Inserting elements ```js // append the script to the body var bodyElem = document.body; bodyElem.appendChild(scriptElem); // insert the div before the first div in the page var firstDiv = document.querySelector('div'); firstDiv.parentNode.insertBefore(divElem, firstDiv); ```
## Event listeners ```js elem.addEventListener('eventName', function(event) { // ... }, false); ```
